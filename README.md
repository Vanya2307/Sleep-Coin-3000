# Sleep Coin 3000

A funny Python Tkinter mini-app that helps decide between sleeping, playing one more game, or getting energy.

## Features

- Dark themed Tkinter GUI
- Coin flip animation
- Time-based decision logic
- Result counters
- Daytime energy mode
- Funny messages and achievements
- Scientific caffeine formula: `E = coffee * milk²`

## Logic

During the day, the app chooses between:

- 🎮 One More Game
- ☕ You Need Energy

At night, the app becomes more concerned and strongly suggests sleep.

## Run

```bash
python sleep_coin.py
